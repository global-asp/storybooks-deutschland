Mi cuerpo Mein Körper My body

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini

Karina Vásquez

Áurea Vericat

Spanisch

Level 1

Con él puedo correr.

Ich kann mit ihm laufen.

I can run with it.

Con él puedo brincar.

Ich kann mit ihm springen.

I can jump with it.

Con él puedo bailar.

Ich kann mit ihm tanzen.

I can dance with it.

Con él puedo nadar.

Ich kann mit ihm schwimmen.

I can swim with it.

Con él puedo saltar.

Ich kann mit ihm hüpfen.

I can skip with it.

Con él puedo patear.

Ich kann mit ihm schießen.

I can kick with it.

Con él puedo escapar.

Ich kann mit ihm weglaufen.

I can escape with it.

Pero con él, nunca podré volar.

Aber ich kann niemals mit ihm fliegen.

But I can never fly with it.
